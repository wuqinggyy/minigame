# 星际闪避者

一个原生 HTML / CSS / JavaScript 写的网页小游戏。

## 玩法

你驾驶一艘小飞船：
- 躲开不断飞来的陨石
- 吃到绿色能量球可以 **加分**、**回血**，并获得短暂无敌护盾
- 生命掉到 0 就结束

## 操作

- `W A S D` 或方向键：移动
- `空格`：在未开始/结束后快速开局
- 点击页面按钮也可以开始或重开

## 运行方法

最简单的方式：
1. 打开 `index.html`
2. 直接开始玩

如果你想用本地静态服务器，也可以在项目目录运行：

```bash
python3 -m http.server 8000
```

然后打开：

```text
http://localhost:8000
```

## 文件说明

- `index.html`：页面结构
- `style.css`：界面样式
- `script.js`：游戏逻辑

## 特点

- 无需后端
- 无需打包工具
- 直接本地可玩
- 带分数、最高分、生命值、开始/重开、结束判定
